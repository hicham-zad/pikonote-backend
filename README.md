How It Works:

1- Définition des variables : L’URL de la page à scraper et le dossier de sortie sont définis.
2- Scraping de la page : La fonction récupère une liste de PDF présents sur la page.
3- Nettoyage du dossier : Le dossier de sortie est supprimé et recréé pour garantir qu’il soit vide.
4- Traitement des PDF : Pour chaque PDF, le fichier est téléchargé et converti en images.
5- Extraction du texte : Chaque image est uploadée sur Cloudinary, puis analysée par OpenAI pour extraire le texte avec la date correspondante.
6- Fusion des textes : Les textes extraits sont triés par date décroissante et combinés en un seul document.
8- Analyse finale et réponse : Le texte fusionné est analysé pour générer une réponse, qui est renvoyée au client en JSON.
